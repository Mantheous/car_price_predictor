# Car Price Predictor

The training data set was found on [kaggle](https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data)

The [[cars.ipynb]] contains the training and EDA process.
[[price_predictor.py]] contains a streamlit app. To use the model to make predictions you can run the following command from the folder of the repo.

```bash
streamlit run price_predictor.py
```

You may need to install streamlit.

```bash
pip install streamlit
```
